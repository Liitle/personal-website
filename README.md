# georgedragan.com

A single-page static site with no build step, no framework, no dependencies. Just an `index.html` deployed on [Vercel](https://vercel.com).

## What's inside

- Work history and current role
- Side projects and open-source work
- Writing / notes
- Contact links

## Running locally

```bash
# Any static file server works, e.g.:
npx serve .
# or
python3 -m http.server
```

Then open `http://localhost:3000` (or whatever port the server prints).
